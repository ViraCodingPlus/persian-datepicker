Persian Datepicker - Calendar Types Example

Year Picker

Calendar that only allows year selection.

انتخاب سال:

Code:


// Year picker example
new PDatepicker('#year-picker', {
    type: 'year',  // Set calendar type to year
    format: 'YYYY' // Only year format
});

Month Picker

Calendar that only allows month and year selection.

انتخاب ماه:

Code:


// Month picker example
new PDatepicker('#month-picker', {
    type: 'month',     // Set calendar type to month
    format: 'YYYY/MM'  // Year and month format
});

Initial Values

Different calendar types with initial values.

سال با مقدار اولیه:

ماه با مقدار اولیه:

Code:


// Year with initial value
new PDatepicker('#year-initial', {
    type: 'year',
    initialValue: '1402/01/01' // Only year part is used
});

// Month with initial value
new PDatepicker('#month-initial', {
    type: 'month',
    initialValue: '1402/06/01' // Year and month parts are used
});

Custom Styling

Custom styled calendar types.

سال با استایل سفارشی:

ماه با استایل سفارشی:

Code:


// Styled year picker
new PDatepicker('#year-styled', {
    type: 'year',
    customStyles: {
        container: {
            boxShadow: '0 10px 25px rgba(74, 107, 255, 0.2)',
            borderRadius: '12px'
        },
        header: {
            backgroundColor: '#4a6bff',
            color: 'white'
        },
        year: {
            padding: '12px',
            borderRadius: '8px',
            transition: 'all 0.2s ease'
        }
    }
});
