Persian Datepicker Examples

This page demonstrates different ways to use the Persian Datepicker in your Laravel projects.

Basic Usage

new PDatepicker('#date1');

Custom Theme

new PDatepicker('#date2', {
    theme: 'dark'
});

Custom Format

new PDatepicker('#date3', {
    format: 'YYYY-MM-DD'
});

Initial Value

new PDatepicker('#date4', {
    initialValue: '1402/06/15'
});

Time Picker

new PDatepicker('#date5', {
    timePicker: true,
    format: 'YYYY/MM/DD HH:mm'
});

Alternate Field

new PDatepicker('#date6', {
    altField: '#date6_alt',
    altFormat: 'YYYY-MM-DD'
});

Event Callbacks

انتخاب نشده
new PDatepicker('#date7', {
    onSelect: function(formattedDate, dateStr) {
        document.getElementById('date7_result').textContent = 
            'تاریخ انتخاب شده: ' + dateStr;
    }
});
